INTRODUCTION TO GIT AND GITHUB

Git is a software used to track different changes made to code base over time Changes can be tracked offline and update are shared by pushing or pulling from a remote repository such as git hub.

Github is a cloud based platform For hosting git repositories and facilitating collaboration.

BASIC GIT COMMANDS

  1. git init
  2. git push
  3. git add .
  4. git commit-m
  5. git status
